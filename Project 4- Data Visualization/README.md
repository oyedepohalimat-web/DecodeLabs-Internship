# Project 4: Data Visualization
# Objective
To develop an interactive dashboard that effectively communicates the key insights, trends and patterns within the dataset.
# Dashboard Features
- Key Performance Indicators (total revenue, total order, average order value, total quantities)
-Interactive Charts
     - Line Chart: Monthly Revenue Trend
     - Bar Chart: Product by Revenue
     - Column Chart: Order Status Distribution
     - Bar Chart: Revenue by Referral Source
     - Donut Chart: Orders by Coupon Code
- Slicers: Payment method, Referral source
# Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
# Skills Demonstrated
- Data Visualization
- Dashboard Design
- Pivot Table Analysis
- KPI Development
- Data Interpretation
- Microsoft Excel
# Outcome
The completed dashboard presents the dataset in a visually appealing and easy-to-understand format. 
It enables users to monitor key metrics, identify trends and explore the data through interactive filters, making it easy to support data-driven decision making.
