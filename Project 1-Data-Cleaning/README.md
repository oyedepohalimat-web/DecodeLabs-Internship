# Project 1: Data Cleaning and Preparation
# Overview
This project focuses on cleaning, preparing and improving the quality of a raw dataset before analysis.
# Objective
To identify and correct data issues that could affect the data analysis results.
# Tasks Performed 
- Examined the raw dataset
- Identified and removed duplicate records
- Checked for missing values
- Corrected inconsistent formatting when necessary
- Prepared the dataset for analysis
# Tools Used
- Power query
- Microsoft Excel
# Skills Demonstrated
- Data Cleaning
- Data Preparation
- Microsoft Excel
# Outcome
A cleaned dataset was successfully prepared and used as the foundation for further analysis.
