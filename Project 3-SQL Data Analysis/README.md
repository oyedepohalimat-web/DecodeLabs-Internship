# Project 3: SQL Data Analysis
# Overview
This project uses SQL to retrieve, manipulate and analyze data from a database to answer business-related questions.
# Dataset Summary 
The database contains structured E-Commerce data stored in relatable tables. SQL queries were used to extract information and generate meaningful insights.
# Objective
To apply SQL techniques to query the database efficiently and answer analytical questions.
# SQL Tasks Performed 
- Retrieved data using SELECT statements.
- Filtered records using WHERE clause.
- Sorted results using ORDERBY.
- Grouped records using ORDERBY.
- Applied aggregate functions such as COUNT, SUM, AVG, MIN, MAX.
# Tools Used
- SQL server
- SQL Server Management Studio (SSMS)
# Skills Demonstrated
- SQL querying
- Database Analysis
- Data Retrieval
- Data Aggregation
- Problem Solving.
# Outcome
The SQL queries successfully extracted valuable and meaningful informations from the database.
