# Project 2: Exploratory Data Analysis
# Overview
This project involved exploring and analyzing the cleaned dataset to identify patterns, trends and important insights.
# Objective
To understand the dataset using descriptive statistics and identify unusual observations using outlier analysis.
# Tools Used
- Microsoft Excel
- Microsoft Word
# Tasks Performed
- Calculated descriptive statistics [count,mean, median, minimum, maximum, sample variance]
- Calculated Quartiles [Q1, Q3] and Inter-quartile Range
- Identified the outliers using the Inter-quartile Range method
- Documented key observations from the analysis 
# Skills Demonstrated
- Descriptive Statistics
- Data Interpretation
- Outlier Detection
- Microsoft Excel
# Outcome
The analysis summarized, highlighted important patterns and also provided insights into the dataset and unusual values.
