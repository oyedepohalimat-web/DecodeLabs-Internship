# PROJECT 1: Data Cleaning and Preparation
# Overview
This project focuses on cleaning, preparing and improving the quality of a raw dataset before anlysis.
# Objective
To identify and correct data issues that could affect analysis results.

- Corrected inconsistent formatting when necessary
- Prepared the dataset for analysis

- Power query
- Microsoft Excel
# Skills Demonstrated
- Data Cleaning
- Data Preparation
- Microsoft Excel
# Outcome
A cleaned and reliable dataset was successfully prepared and used as the foundation for further analysis.
